# Vanilla SSE

## Project Structure

```
vanila-sse/
├── src/
│   ├── server.js     # Node.js SSE server
│   ├── client.js     # Browser-side SSE client
│   └── client.html   # Client HTML page
└── package.json
```

## Usage

```bash
# Install dependencies
pnpm install

# Start the server
pnpm run start:server
```
